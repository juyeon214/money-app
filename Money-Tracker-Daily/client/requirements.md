## Packages
date-fns | Date manipulation and formatting for the calendar
recharts | Data visualization for the analytics graph tab
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility to merge Tailwind classes without style conflicts
lucide-react | Beautiful, consistent icons

## Notes
- App uses a Teal/Mint primary color theme for a modern financial feel.
- Relies on 'Noto Sans KR' for Korean typography.
- Uses `sessionStorage` or React Context for persisting the selected month across tabs.
